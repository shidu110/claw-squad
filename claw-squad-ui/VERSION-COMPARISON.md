# ClawSquad 双版本 UI 对比

## 版本 A: Tech/Space (暗色科技风)
**文件**: `src/TeamAApp.tsx`

### 设计特点
- 暗色背景 (Slate-900)
- 渐变色标题 (Indigo → Purple)
- 发光边框效果
- 霓虹风格图标
- 深色终端

### 适用场景
- 开发者工具
- 技术展示
- 专业 IDE 风格

---

## 版本 B: Minimal/Clean (简洁商务风)
**文件**: `src/TeamBApp.tsx`

### 设计特点
- 浅色背景 (Gray-50)
- 纯色标题 (Blue)
- 简洁卡片
- 清晰层次
- 经典终端

### 适用场景
- 企业应用
- 普通用户
- 快速上手

## 对比表

| 维度 | Team A (Tech) | Team B (Minimal) |
|------|---------------|------------------|
| **风格** | 暗色科技 | 简洁商务 |
| **背景** | Slate-900 | Gray-50 |
| **主色** | Indigo/Purple | Blue |
| **边框** | 渐变发光 | 纯色线条 |
| **图标** | Lucide (渐变) | Lucide (纯色) |
| **终端** | 深色 (Slate-950) | 深色 (Gray-900) |
| **状态** | 霓虹点状 | 纯色点状 |
| **适用** | 开发者 | 企业/通用 |

---

## 切换方式

```typescript
// main.tsx
import TeamAApp from './TeamAApp'  // Tech/Space
import TeamBApp from './TeamBApp'  // Minimal/Clean

// 选择一个
<TeamAApp />  // 或 <TeamBApp />
```

## 建议

**推荐 Team A (Tech/Space)** 作为默认：
- 更具视觉冲击力
- 区分度更高
- 符合 AI/技术产品调性

**保留 Team B (Minimal)** 作为备选：
- 可作为"商务模式"主题
- 企业用户友好
