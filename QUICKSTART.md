# ClawSquad Quick Start

## One-Command Setup

bash <(curl -fsSL https://raw.githubusercontent.com/USER/ClawSquad/main/setup.sh)

## Manual Setup

1. Clone repo
2. Run: npm install
3. Configure API: npm run config
4. Start: npm start
