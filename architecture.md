# ClawSquad Architecture

## Modules

1. **claw-squad-core** - CEO Brain, Team Factory
2. **claw-squad-cli** - CLI Orchestrator
3. **claw-squad-competition** - Referee, Quality Scorer
4. **claw-squad-ui** - React Visualization
5. **claw-squad-mcp** - OpenClaw Bridge

## Data Flow

Task -> CEO Brain -> Team Factory -> [Team A vs Team B] -> Referee -> Winner
